# Jot.zone
Code for web/iOS/Android Jot.zone app

## Built with

* [React Native](https://reactnative.dev/)
* [React Native Web](https://necolas.github.io/react-native-web/)
* [Google Firebase](https://firebase.google.com/)

## Run locally

Install,

```
npm install
```

Run,

* Web: `npm run web`
* Android: `npm run android`
* iOS: `npm run iOS`
